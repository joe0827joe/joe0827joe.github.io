# 許應良 - AI 工程師履歷

這是我的個人履歷網站，展示專業技能、專案作品與工作經歷。

## 技術棧
- HTML5 / CSS3 / JavaScript
- Font Awesome / Google Fonts

## 內容架構
| 層次 | 用途 | 網址 |
|------|------|------|
| 作品集網站 | 形象、架構圖、完整專案說明 | https://joe0827joe.github.io/ |
| 一頁式履歷 | ATS／投遞／轉寄／列印 PDF | https://joe0827joe.github.io/resume.html |
| 共用資料 | 網站與履歷同源內容 | `data/resume.json` |

更新履歷時，優先修改 `data/resume.json`，一頁履歷會自動同步；作品集頁的長文與視覺證據仍在 `index.html`。

## 匯出 PDF
1. 開啟 [一頁履歷](https://joe0827joe.github.io/resume.html)
2. 點「下載 PDF」，於列印對話框選擇「另存為 PDF」
3. 或使用 `resume.html?print=1` 直接叫出列印
